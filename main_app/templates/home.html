Ansible + Django

Ansible is an open-source software provisioning, configuration management, and application-deployment tool enabling infrastructure as code. and Windows.

Django is a high-level Python web framework that enables rapid development of secure and maintainable websites.

This application tries to combine simplicity of web interface and power of automation tools so even non technical users can leverage these tools in their work.

  • Features
  • Add/Manage Server
  • Create/Manage Ansible Groups
  • Create/Manage Inventory
  • Deploy upgrade using web based wizard ui
  • Manage upgradable packages
